# ProxAI Documentation

ProxAI is a small local compatibility proxy for OpenAI-compatible requests. It accepts local OpenAI-compatible traffic, normalizes specific upstream-breaking request shapes, routes to explicit providers, and translates between supported protocols when configured.

<ProxaiCallout type="tip" title="Focused by design">
  ProxAI is intentionally small: local compatibility, explicit protocol routing, compact diagnostics, and minimal surprises. It is not a general multi-tenant AI gateway.
</ProxaiCallout>
