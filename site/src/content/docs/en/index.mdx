# ProxAI Documentation

ProxAI is a small local compatibility proxy for OpenAI-compatible requests. It normalizes a few upstream-breaking request shapes, routes to explicit providers, and translates across supported protocols when configured.

<ProxaiCallout type="tip" title="Focused by design">
  ProxAI is intentionally small: local compatibility, explicit protocol routing, compact diagnostics, and minimal surprises.
</ProxaiCallout>
