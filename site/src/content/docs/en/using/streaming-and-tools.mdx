# Streaming and Tools

Use this page when streaming works for plain text but fails, stalls, or returns unexpected events when tools are involved.

<ProxaiCallout type="note" title="There are two timeout layers">
  `read_idle_timeout_secs` watches upstream bytes. `[tool_calls].timeout_secs` watches semantic completion of streamed tool-call arguments. They solve different problems.
</ProxaiCallout>
