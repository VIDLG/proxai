# Troubleshooting

Start from the symptom the client sees, then inspect the narrowest useful layer. Each card gives likely causes, first checks, a useful capture phase, and the docs that own the deeper explanation.

<ProxaiCallout type="tip" title="Capture before guessing">
  Most symptoms are easiest to diagnose with one sanitized capture. Enable the narrowest useful phase for a single request, then disable it again. See [Capture Phases](/en/reference/capture-phases).
</ProxaiCallout>
