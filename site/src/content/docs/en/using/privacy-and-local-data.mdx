# Privacy and Local Data

ProxAI is a local compatibility proxy. That makes it useful for debugging, but it also means local files can contain provider configuration, request metadata, captured payloads, and diagnostics.

<ProxaiCallout type="caution" title="Treat captures as sensitive">
  Captures can contain prompts, tool arguments, provider output, file snippets, model names, and request timing. Do not commit or share them without review and redaction.
</ProxaiCallout>
