# Behavior Contracts

These are promises ProxAI commits to. Breaking any of them is a breaking change that requires a major version bump and a migration note (see [C30](#versioning)). They are more stable than field names or config defaults.

<ProxaiCallout type="note" title="Contracts are not tutorials">
  Contracts are the stable behavior reference. Task pages should link here instead of restating these promises in full.
</ProxaiCallout>
