# Configuration Reference

This page is the **single-source configuration reference** for the tracked `config.example.toml`. ProxAI writes this file into the app directory on first run, then loads your local `config.toml` for actual runtime settings.

<ProxaiCallout type="caution" title="Never commit secrets">
  Treat `config.example.toml` as a template. Put real `api_key` values only in your local git-ignored `config.toml`.
</ProxaiCallout>
