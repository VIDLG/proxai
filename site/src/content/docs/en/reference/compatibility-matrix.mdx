# Compatibility Matrix

This page is the stable lookup table for protocol-pair behavior. Use [Choosing a Protocol](/en/protocol/choosing-a-protocol) for guidance and [Lossiness and Fidelity](/en/protocol/lossiness-and-fidelity) for interpretation.

<ProxaiCallout type="note" title="Supported does not mean lossless">
  A supported pair has explicit conversion behavior. It can still be lossy where the target protocol cannot express a source-protocol detail.
</ProxaiCallout>
