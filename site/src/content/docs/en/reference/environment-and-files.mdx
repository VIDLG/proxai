# Environment and Files

ProxAI keeps runtime state local. The repository contains source code and sanitized examples; the app directory contains runtime configuration, logs, and captures.

## App directory

<ReferenceTable
  columns={['Platform', 'Default app directory']}
  rows={[
    ['Windows', '`%USERPROFILE%\\.proxai\\`'],
    ['Linux/macOS', '`~/.proxai/`'],
  ]}
  codeFirstColumn={false}
/>

<ProxaiCallout type="caution" title="Keep local artifacts local">
  Do not commit `config.toml`, API keys, private captures, logs, or local repro fixtures containing private prompts. Committed fixtures should be trimmed and sanitized.
</ProxaiCallout>
