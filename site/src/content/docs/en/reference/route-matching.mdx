# Route Matching

Route matching selects a provider for one inbound request. Provider `protocol` then decides the outbound wire behavior.

## Matching outcomes

<ReferenceTable
  columns={['Condition', 'Outcome']}

<ProxaiCallout type="tip" title="Use request_protocol sparingly">
  Omit `request_protocol` for ordinary model routing. Set it only when the same model pattern needs endpoint-specific behavior.
</ProxaiCallout>
