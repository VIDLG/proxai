# Test Map

Use this page after [Where to Change Code](/en/developer/where-to-change-code) to choose the smallest validation that covers a change, then expand outward only when needed.

<ProxaiCallout type="note" title="Prefer just recipes on Windows">
  On Windows, prefer project `just` recipes when a local proxy may be running. They avoid target-directory conflicts with a locked `target/debug/proxai.exe`.
</ProxaiCallout>

## Common commands

```sh
just site check
cargo fmt --check
pixi run cargo clippy --lib --tests -- -D warnings
pixi run cargo test --lib
just test-e2e
just check
```

Use the exact project recipes from the root `justfile` and `site/justfile`; do not invent new validation commands in docs unless they are added to the repository.
