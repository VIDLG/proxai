# Developer Guide

This section is for maintainers changing ProxAI internals. It focuses on source-level boundaries, protocol conversion rules, streaming behavior, and error projection.

<ProxaiCallout type="caution" title="Internal docs">
  These pages describe implementation boundaries. User-facing behavior belongs in [Using ProxAI](/en/using), [Protocol Guide](/en/protocol), and [Reference](/en/reference).
</ProxaiCallout>
