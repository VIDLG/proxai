# Protocol Overview

ProxAI separates **where data is in the proxy pipeline** from **which wire protocol that data uses**. This page is the protocol guide hub: start here when you need to understand request paths, protocol values, conversion support, or per-protocol walkthroughs.

    ['anthropic_messages', 'Anthropic Messages API: single `Message`, flat `content[]` blocks, `content_block` streaming events.'],
  ]}
/>

<ProxaiCallout type="note" title="Routing is not translation">
  A route may select a provider whose protocol differs from the inbound request protocol. That requires an explicit translation pair; unsupported pairs fail explicitly.
</ProxaiCallout>
