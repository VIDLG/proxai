# Chat Completions Interaction Example

Back to [OpenAI Chat Completions](/en/protocol/openai-chat-completions).

This compact example mirrors the detailed Chinese walkthrough. It shows the Chat shape: candidate `choices[]`, each containing one assistant delta/message and optional `tool_calls[]`.

<EventTimeline events={[
  { name: 'client request', description: 'Client sends ordered `messages`, a function tool, optional search-like request settings, and `stream: true`.' },
  { name: 'assistant delta', description: 'The stream starts with a choice delta, usually carrying `role: assistant`.' },
  { name: 'tool call deltas', description: '`choices[].delta.tool_calls[]` carries function argument fragments.' },
  { name: 'finish_reason', description: '`finish_reason: tool_calls` marks that arguments are complete and local tools should run.' },
  { name: 'tool message', description: 'The next request sends a `role: tool` message with the matching `tool_call_id`.' },
  { name: 'final answer', description: 'The final stream emits `delta.content`, then `finish_reason: stop`, then `[DONE]`.' },
]} />
