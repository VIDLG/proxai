# ProxAI 文档

ProxAI 是一个本地 OpenAI 兼容代理：接收本地 OpenAI-compatible 请求，修复部分上游不兼容的请求形态，按显式配置路由到 provider，并在已支持的协议对之间做转换。

<ProxaiCallout type="tip" title="保持聚焦">
  ProxAI 的边界是本地兼容、显式协议路由、紧凑诊断和尽量少的意外；不是通用多租户 AI 网关。
</ProxaiCallout>
