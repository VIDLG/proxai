# 协议模型总览

ProxAI 把**数据处在代理链路的哪个阶段**和**这个阶段使用哪种 wire protocol**拆成两个独立概念。这个页面是协议指南的入口：当你需要理解 request path、协议取值、转换支持或各协议交互示例时，从这里开始。

</ProxaiCallout>
