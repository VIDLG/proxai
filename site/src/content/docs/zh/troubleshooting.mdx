# 常见排障

本页汇总 ProxAI 最常见的症状，并指向正确的下一步。这里不重复解释协议内部或配置字段；每张卡片都链接到负责该内容的页面。

<ProxaiCallout type="tip" title="先 capture，再猜测">
  大多数症状用一次脱敏 capture 最容易诊断。参考 [观测与诊断](/zh/observability)，为单次请求开启 `inbound_request`、`provider_request`、`upstream_response` 或 `outbound_response`，调试完再关掉。
</ProxaiCallout>
