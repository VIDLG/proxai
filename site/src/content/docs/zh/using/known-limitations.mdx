# 已知限制

这页用于说明 ProxAI 今天有意不做什么。它是面向用户的摘要；精确协议支持见 [兼容性矩阵](/zh/reference/compatibility-matrix)，精确配置字段见 [配置参考](/zh/reference/configuration)。

<ProxaiCallout type="note" title="限制也是设计边界">
  ProxAI 是小型本地兼容代理。除非项目明确选择这个方向，否则它不应扩展成通用 AI gateway。
</ProxaiCallout>

## 如果遇到限制

1. 先在对应 reference 页面确认当前行为。
2. 用最小请求和最窄有用 capture phase 复现。
3. 判断变更属于 provider compatibility、protocol translation、routing 还是 docs。
4. 如果会改变用户可见行为，同时更新用户文档和 reference 文档。

本地调试路径见 [调试工作流](/zh/using/debugging-workflow) 和 [常见排障](/zh/using/troubleshooting)。
