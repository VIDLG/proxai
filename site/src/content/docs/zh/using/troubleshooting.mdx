# 常见排障

从客户端看到的症状开始，再检查最窄的有用层级。每张卡片都给出可能原因、第一批检查项、有用的 capture phase，以及深入解释所在文档。

<ProxaiCallout type="tip" title="先 capture，再猜测">
  大多数症状用一次脱敏 capture 最容易诊断。为单次请求开启最窄的有用 phase，调试完再关掉。见 [Capture Phases](/zh/reference/capture-phases)。
</ProxaiCallout>
