# 流式与工具调用

当纯文本 streaming 正常，但涉及工具时失败、卡住或返回意外事件，看本页。

<ProxaiCallout type="note" title="有两个 timeout 层级">
  `read_idle_timeout_secs` 观察上游字节。`[tool_calls].timeout_secs` 观察流式工具调用参数的语义完成。它们解决不同问题。
</ProxaiCallout>
