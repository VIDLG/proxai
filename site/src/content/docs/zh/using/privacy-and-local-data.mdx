# 隐私与本地数据

ProxAI 是本地兼容代理。这让它适合调试，但也意味着本地文件可能包含 provider 配置、请求元数据、capture payload 和诊断信息。

<ProxaiCallout type="caution" title="把 capture 当作敏感数据">
  Capture 可能包含 prompt、工具参数、provider 输出、文件片段、模型名和请求时间。未经审查和脱敏，不要提交或分享。
</ProxaiCallout>
