# 测试地图

先用 [该改哪里](/zh/developer/where-to-change-code) 定位源码边界，再用本页选择覆盖该变更的最小验证；只有必要时再扩大范围。

<ProxaiCallout type="note" title="Windows 上优先使用 just recipes">
  在 Windows 上，如果本地 proxy 可能正在运行，优先使用项目 `just` recipes。它们可以避免被锁定的 `target/debug/proxai.exe` 导致 target 目录冲突。
</ProxaiCallout>

## 常用命令

```sh
just site check
cargo fmt --check
pixi run cargo clippy --lib --tests -- -D warnings
pixi run cargo test --lib
just test-e2e
just check
```

使用根目录 `justfile` 和 `site/justfile` 里真实存在的项目 recipes；不要在文档里发明尚未加入仓库的新验证命令。
