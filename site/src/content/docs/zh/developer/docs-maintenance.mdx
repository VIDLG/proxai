# 文档维护

本页定义 ProxAI 文档应如何演进。它面向维护者，并补充 `AGENTS.md`。

<ProxaiCallout type="caution" title="文档跟随代码">
  文档跟随代码和已验证行为；代码不跟随愿景式文档。当文档和实现不一致时，先检查实现、测试和配置默认值，再有意识地更新文档或代码。
</ProxaiCallout>

## 验证

发布文档前运行站点检查：

```sh
just site check
```

该命令会构建 Starlight 站点，并检查双语页面成对、sidebar slug、内部链接、锚点、旧路径、Markdown 表格残留、developer `noindex`、hub 覆盖和标题质量。
