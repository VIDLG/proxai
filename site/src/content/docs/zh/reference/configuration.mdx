# 配置参考

本页是被跟踪的 `config.example.toml` 的**单一来源配置参考**。ProxAI 首次运行时会把该文件写到应用目录，然后从本地 `config.toml` 加载真实运行时配置。

  把 `config.example.toml` 当模板。真实的 `api_key` 只放在本地、不进入 git 的 `config.toml` 里。
</ProxaiCallout>
