# 环境与文件

ProxAI 把运行时状态保留在本地。仓库包含源码和脱敏示例；app 目录包含运行时配置、日志和 captures。

    ['Linux/macOS', '`~/.proxai/`'],
  ]}
  codeFirstColumn={false}
/>

<ProxaiCallout type="caution" title="本地产物留在本地">
  不要提交 `config.toml`、API keys、private captures、logs，或包含 private prompts 的本地复现 fixtures。提交的 fixtures 应该裁剪并脱敏。
</ProxaiCallout>
